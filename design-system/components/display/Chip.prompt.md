Small labelled token — view tags, DPI/method badges, status pills, comment roles, out-of-zone warnings. Tonal palette maps to the semantic intents.

```jsx
<Chip>front</Chip>                              {/* neutral soft */}
<Chip tone="emerald" variant="solid">Согласовано</Chip>
<Chip tone="blue">Клиент</Chip>
<Chip tone="red" size="sm">за зоной</Chip>
<Chip tone="amber" variant="soft">150 dpi</Chip>
```

`variant`: `soft` (dark tinted, default), `solid` (full color), `outline`. Sizes `sm` / `md` / `lg`.
