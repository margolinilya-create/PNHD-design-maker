Inline Lucide icon (the system's single icon set, thin 1.75 stroke). Inherits color from `currentColor`. Requires the Lucide UMD script loaded once on the page.

```jsx
<Icon name="upload" />
<Icon name="trash-2" size={18} />
<Icon name="chevron-left" style={{ color: "var(--text-label)" }} />
```

Use kebab-case Lucide names. Pair with `IconButton` for clickable affordances.
