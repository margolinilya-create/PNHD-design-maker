Labeled text or number input — dark inset (gray-900) with a blue focus ring. For metric entry use `numeric` (tabular figures) and a unit `suffix`.

```jsx
<Field label="Клиент" placeholder="Название" />
<Field label="Ширина" numeric suffix="мм" type="number" defaultValue={120} />
<Field label="Заказ №" hint="печатается в обвязке" />
```
